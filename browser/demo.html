🚀 Getting Started

  1. Start the Web-IDE-Bridge server: npm start in the server directory
  2. Launch the Web-IDE-Bridge desktop application
  3. Click "Connect to Server" below to establish the connection
  4. Click "Edit in IDE ↗" next to any textarea to open it in your preferred IDE
  5. Make changes in your IDE and save (Ctrl+S/Cmd+S) to see them sync back instantly
